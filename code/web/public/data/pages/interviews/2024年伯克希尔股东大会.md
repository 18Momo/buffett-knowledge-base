## 详细摘要

### 第二部分

问题是这样的：在去年的一次采访中，托德·康姆斯(ToddCombs)表示，2010年刚加入伯克希尔时，他对您说过，[[盖可保险]](GEICO)以营销和[[品牌]]见长，而前进保险(Progressive)在数据分析方面更胜一筹，从长期来看，数据占优者将从竞争中胜出。像[[盖可保险]]这样的公司，经过几十年的发展，可能有需要转型的时候，而伯克希尔一直采取放手的管理方式，这样的话，是否容易延误转型的时机？比如说，有人给我发过来一段视频说，我是你女儿，刚出了车祸，需要5万美元，赶快打给我。

### 第三部分

关于人生建议，查理讲过，“反着来，想想自己死的时候要得到怎样的结果，按照那样的结果倒推着生活。把时间倒退几个世纪，世界一成不变，你是放羊的，100年后，你的孙子孙女还是放羊的。过去200年里，随着工业革命的发生，科学、教育、健康等方方面面取得了突飞猛进的发展。

### **下午场**

伯克希尔是一家上市公司，也拥有巨额财富，我们的不同之处在于，我们的股东遍布美国各地，不仅仅是在奥马哈，大量伯克希尔股东，他们通过单独的个人行为，默默地捐赠了1亿美元以上的资金，支持本地慈善事业的发展。特别巧的是，就在我们买入露丝那10亿美元伯克希尔A类股当天，我们的财务总监马克·米勒德(MarkMillard)还从另一位股东手中买入了5亿美元的股票。美国有很多财力雄厚的上市公司，有的上市公司的控股股东为某个家族，他们积聚了大量财富，并且将财富用于慈善事业，例如，沃尔玛的沃顿家族，还有微软的比尔·盖茨也可以归为此类。

### 第五部分

与查理第一次见面的时候，我对西海岸的众多小公司如数家珍，查理对我刮目相看。特拉华-拉克瓦纳-西部铁路(Delaware,Lackawanna&WesternRailroad)设计了一个名为“白菲比”(PhoebeSnow)的代言人，她身着一袭白衣，宣传无烟煤更清洁，人们就管这条铁路叫“白菲比”。总之，今天重新再来，我仍然会翻遍所有小公司，把每一家小公司挨个看、研究透(tryandknoweverythingabouteverythingsmall)。

### **上午场**

按照姓名字母顺序，出席本次股东会的董事有：霍华德·巴菲特(HowardBuffett)、苏茜·巴菲特(SusieBuffett)、史蒂夫·伯克(SteveBurke)、肯·切诺特(KenChenault)、克里斯·戴维斯(ChrisDavis)、苏珊·德克尔(SueDecker)、夏洛特·盖曼(CharlotteGuyman)、小汤姆·墨菲(TomMurphyJr.)、罗恩·奥尔森(RonOlson)、沃利·韦茨(WallyWeitz)、梅丽尔·威特默(MerylWitmer)。首先，我要感谢梅利莎·夏皮罗(MelissaShapiro)，她是此次大会的组织者，她为筹办本次股东会付出了巨大的努力。出席本次股东大会的还有台上在座的[[格雷格·阿贝尔]](GregAbel)、阿吉特·贾因(AjitJain)。

### 具体说到比亚迪这笔投资……

考虑到将来可能被政府[[收购]]，伯克希尔·哈撒韦能源还会继续向公用事业公司投资吗？新政策规定，犹他州的发电厂所生产的电力只能由州政府独家购买，而且犹他州政府有权在发电厂关停之前，对发电厂进行[[收购]]。依据新的立法，犹他州政府有权优先[[收购]]电力和电厂，其中包括伯克希尔·哈撒韦能源下属的落基山电力公司(RockyMountainPower)。
