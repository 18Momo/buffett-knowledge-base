# 1968年11月 巴菲特致合伙人信

## 核心要点

- 我们今年和往年不一样，大部分时间都持有大量可以作为现金等价物的证券，而且有价值的机会星星点点，但是无论按照新标准还是旧标准衡量，我们的业绩都相当好。
- 1961年财政部寄发给我们的税收信函规定我们是合伙性质，纳税地位有别于公司法人，另外，根据我们的合伙基金合同，以及联邦所得税对合伙性质纳税实体的一般规定，都不允许将合伙基金权益转移到信托。
- 有些合伙人已经是我的老朋友了，我很难拒绝他们，但是允许新成员加入，会损害所有现有合伙人的利益，而且我也不愿意区别对待各位合伙人，答应这个，不答应那个。
- 道指目前上涨了8%左右，年中时我们领先道指15个百分点，现在我们领先的要更多一点。
- ”这个规定没有例外，我没办法，只能把现有合伙人的配偶、儿子、女儿、孙子、孙女等等都拒绝了（“等等”二字省略的内容，请各位自行补充）。
- 再提醒各位一下，请尽快将承诺书寄给我们，看谁能赢得“年度合伙人”大奖（这是个荣誉奖，没奖金）。

## 详细摘要

### 年度概览

我们今年和往年不一样，大部分时间都持有大量可以作为现金等价物的证券，而且有价值的机会星星点点，但是无论按照新标准还是旧标准衡量，我们的业绩都相当好。1961年财政部寄发给我们的税收信函规定我们是合伙性质，纳税地位有别于公司法人，另外，根据我们的合伙基金合同，以及联邦所得税对合伙性质纳税实体的一般规定，都不允许将合伙基金权益转移到信托。有些合伙人已经是我的老朋友了，我很难拒绝他们，但是允许新成员加入，会损害所有现有合伙人的利益，而且我也不愿意区别对待各位合伙人，答应这个，不答应那个。
