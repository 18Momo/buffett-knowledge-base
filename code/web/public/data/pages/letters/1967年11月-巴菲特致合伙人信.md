# 1967年11月 巴菲特致合伙人信

## 核心要点

- 大家在解读我们今年剩下两个月甚至是明年的业绩时，有一点请各位知悉：我们现在不但在控股公司中投资了约2000万美元，而且还有1600多万美元买了短期国债。
- 这不是因为我对市场走向有什么判断，只是我找不到任何很明显能赚钱、又安全的机会（从长期价值的角度考虑，不是短期的市场报价），没地方投资这些钱。
- 一份“基本原则”，其中第四条和第五条完全修改了，这是因为现在的环境变了，具体已经在10月9日的信中讲过了。
- 在将承诺书寄给我们之前，请务必仔细了解自己的纳税情况，免得明年4月份纳税时出问题。
- 也就是说，如果股市继续上涨，现在我们总共有3600万美元，都不会参与其中。
- 一位合伙人给我们提了一条很好的建议，我们就按他说的，把副本也寄给大家。
- 如果这个目标，就算实现了，你也不满意，你最好在年末把自己的资金提出去。

## 详细摘要

### 年度概览

大家在解读我们今年剩下两个月甚至是明年的业绩时，有一点请各位知悉：我们现在不但在控股公司中投资了约2000万美元，而且还有1600多万美元买了短期国债。这不是因为我对市场走向有什么判断，只是我找不到任何很明显能赚钱、又安全的机会（从长期价值的角度考虑，不是短期的市场报价），没地方投资这些钱。一份“基本原则”，其中第四条和第五条完全修改了，这是因为现在的环境变了，具体已经在10月9日的信中讲过了。

## 原文金句

> 大家在解读我们今年剩下两个月甚至是明年的业绩时，有一点请各位知悉：我们现在不但在控股公司中投资了约2000万美元，而且还有1600多万美元买了短期国债。

> 这不是因为我对市场走向有什么判断，只是我找不到任何很明显能赚钱、又安全的机会（从长期价值的角度考虑，不是短期的市场报价），没地方投资这些钱。

> 在将承诺书寄给我们之前，请务必仔细了解自己的纳税情况，免得明年4月份纳税时出问题。

> 也就是说，如果股市继续上涨，现在我们总共有3600万美元，都不会参与其中。

> 一位合伙人给我们提了一条很好的建议，我们就按他说的，把副本也寄给大家。

> 往年，新的一年开始时，我们总是有一些就要孵出小鸡的蛋，今年，到现在鸡窝里还空空如也。
