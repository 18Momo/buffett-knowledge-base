## 详细摘要

### 保险承保业务

在1974-75年危机期间众多竞争对手收缩或退出之后，国民保险不仅实现了保费收入的大幅增长，而且保持了优异的承保利润率。这家工伤赔偿保险公司是在1977年最后几天以现金[[收购]]的，因此其当年约1,250万美元的保费收入未纳入我们的业绩。不过在此期间，国民保险的承保盈利能力已经大幅提升，同时还积累了大量可供投资的资金。

### 保险投资业务

过去两年里，[[保险业]]务的投资成本（不含对关联公司[[蓝筹印花]]的投资）已从1.346亿美元增长到2.528亿美元。1977年12月31日，我们旗下保险公司持有的市值超过500万美元的股票投资如下：持股数量公司名称成本（千美元）市值（千美元）保费收入大幅增长带来的保险准备金增加，加上[[留存收益]]，共同推动了有价证券投资规模的扩大。

### 年度概览

一方面期初资本比一年前又增加了23%，另一方面我们预计保险承保利润率会在年底之前开始走下坡。1977年，我们以期初股东权益计算的经营回报率约为19%，比上一年略好，也高于我们自身的长期平均水平及美国企业的整体平均水平。除非是特殊情况（例如负债比例异常高的公司，或是资产负债表上有重要资产被严重[[低估]]的公司），我们认为股本回报率才是衡量[[管理层]]经济表现更合理的指标。

### 银行业务

因此，前奥马哈美国国民银行(AmericanNationalBankofOmaha)总裁兼首席执行官彼得·杰弗里(PeterJeffrey)已于3月1日正式加入伊利诺伊国民银行，担任总裁兼首席执行官。去年末，现年80岁、仍然在一线经营着这家无人能及的银行的吉恩，提出希望引进一位接班人。我们有理由期待罗克福德(Rockford)地区这家首屈一指的银行继续保持成功。

### 纺织业务

这或许说明了我们预测能力的薄弱，或者说明了纺织行业的本质，也许两者兼而有之。营销上面临的许多困难主要源于行业大环境，但也有不少问题是我们自己造成的。
