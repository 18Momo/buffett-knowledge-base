## 原文金句

> 如果你迟迟不进场，就犯了一个非常可怕的错误，错失了从时间跨度上分散风险的机会。

> **在1960年左右，PhilFisher写过一本投资著作，名字叫《怎样选择成长股》（CommonStocksandUncommonProfits），我建议投资者都该读读这本书，从Fisher那里我了解到了"聊天法"（scuttlebutt）的价值：走出去，和竞争对手、供应商、客户们交谈，彻底弄清楚一个行业或者一家公司实际上是如何经营运作的。

> 再看未来[[苹果]]的盈利能力，我认为库克做了一件了不起的工作，他的[[资本配置]]非常聪明。

> 现在，我们[[富国银行]]持股就超过10%，不过这些都是因为银行自己[[回购]]股份，并不是我们买进股票。

> **CNBC：**在一周前，我们第一次听说了卡夫[[收购]]联合利华的消息，但是周五的时候一条新闻推翻了此前的消息，卡夫宣布撤回[[收购]]联合利华要约。

> **我自己做[[收购]]不是这样的，我就进去然后说，"如果让我给报价，那么我就做一个，如果你不让我做报价，我就不做。
