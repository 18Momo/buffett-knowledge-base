# 1965年11月 巴菲特致合伙人信

## 核心要点

- 基本原则：一笔投资，我们掌握了事实和逻辑，正确的概率极大，与此同时，这笔投资的[[内在价值]]不可能出现根本变化，出错的概率很小，在这种情况下，我们可能最多拿出40%的净资产用于这笔投资。
- 基本原则：如果我们的长期收益率高于每年6%，合伙人的盈利金额会大于提现金额，合伙人的本金会增加。
- 基本原则：2.我们在每一笔投资中都追求极大的[[安全边际]]并[[分散投资]]，力图将永久性资本损失（不是短期账面亏损）的风险降到绝对最小值。
- 鉴于应收账款和存货的性质，在上述估值方法中，流动资产以[[账面价值]]计算，固定资产以50%的折价率计算。
- 这个估值对追加投资和提取现金的合伙人都公平，它可能高于也可能低于市值。
- 今年年末，我们对[[伯克希尔哈撒韦]]公司控股权益的估值将介于净流动资产价值和[[账面价值]]二者之间。
- 哪怕是再保守老派的合伙人，即使使用净利润这样原始的衡量标准，也会对我们的业绩非常满意。

## 详细摘要

### 基本原则

一笔投资，我们掌握了事实和逻辑，正确的概率极大，与此同时，这笔投资的[[内在价值]]不可能出现根本变化，出错的概率很小，在这种情况下，我们可能最多拿出40%的净资产用于这笔投资。如果我们的长期收益率高于每年6%，合伙人的盈利金额会大于提现金额，合伙人的本金会增加。2.我们在每一笔投资中都追求极大的[[安全边际]]并[[分散投资]]，力图将永久性资本损失（不是短期账面亏损）的风险降到绝对最小值。
