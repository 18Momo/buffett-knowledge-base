# 1962年11月 巴菲特致合伙人信

## 核心要点

- 至于资产变现的成果，或许用一句话就能说清楚：1961年11月30日（登普斯特财政年度末），我们拥有166,000美元的现金，却有2,315,000美元的负债；而今年底，预计将有大约100万美元的现金及投资（投资方式与合伙基金一致），负债只剩25万美元。
- 在这22.3个百分点中，约60%来自投资组合中登普斯特以外的部分，40%来自登普斯特估值的提升。
- 1963年有望继续改善，我们预计明年登普斯特的投资组合规模将大幅扩张。
- 举个例子，去年底4,200,000美元的存货，今年底大概只剩1,900,000美元左右——仅此一项，折价损失就减少了约92万美元（减少额230万美元的40%）。

- 增资款必须在1月10日前到账；如果在11月就汇来，将按预付款处理，并从收款之日起按6%的年利率计算利息，直至年末。
- 它是一种方便的定期提现安排——若我们的年收益率超过6%，这笔钱来自盈利；若达不到6%，则全部或部分是本金的返还。
