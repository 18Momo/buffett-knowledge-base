# 1959 巴菲特致合伙人信

## 核心要点

- 沃伦·E·巴菲特：不过我们认为，Tri-Continental这样的机构受托管理的是股东的资金，承担风险是有边界的——我们相信目前的投资组合已经为来年做好了准备。
- 沃伦·E·巴菲特：1959年这样的市场，投资者的情绪与热情主导一切，受过价值投资训练、以长期视角操作的基金经理在这种环境里步履维艰。
- 沃伦·E·巴菲特：目前各账户的投资组合相似度约达到80%，之所以还有差异，是因为各账户的现金投入时间、股票买入时机、向合伙人支付利息等情况各有不同。
- 沃伦·E·巴菲特：总裁弗雷德·布朗(FredBrown)最近在分析师协会的演讲中，对1959年的市场这样说道："尽管我们非常看好自己的投资组合，但Tri-Continental的持仓在1959年的市场表现还是让我们失望了。
- 沃伦·E·巴菲特：去年道指从583点涨到679点，涨幅16.4%，加上[[股息]]，持有道指的投资者全年总收益率为19.9%。
- 沃伦·E·巴菲特：但按照传统估值标准来看，当前蓝筹股的股价中包含了相当大的投机成分，对应着相应的亏损风险。
- 沃伦·E·巴菲特：计算时已扣除经营费用，但未扣除按合伙协议约定支付给合伙人的利息，也未扣除支付给普通合伙人的利润分成。

## 详细摘要

### 沃伦·E·巴菲特

不过我们认为，Tri-Continental这样的机构受托管理的是股东的资金，承担风险是有边界的——我们相信目前的投资组合已经为来年做好了准备。1959年这样的市场，投资者的情绪与热情主导一切，受过价值投资训练、以长期视角操作的基金经理在这种环境里步履维艰。目前各账户的投资组合相似度约达到80%，之所以还有差异，是因为各账户的现金投入时间、股票买入时机、向合伙人支付利息等情况各有不同。

### 当前投资组合

在写这封信的时候，道指已经回吐了1959年涨幅的一半以上——今年也许就是检验熊市表现那个预测的时候了。在价值最终实现之前，这只股票大概率会跑赢大盘，而我希望今年就能看到这一天。我们是以相对于资产价值大幅折价的价格买入的——这个折价是依据他们持有股票的市场价值，加上对其主营业务的保守估值来计算的。
