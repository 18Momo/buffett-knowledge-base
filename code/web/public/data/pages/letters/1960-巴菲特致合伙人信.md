# 1960 巴菲特致合伙人信

## 核心要点

- 五、桑伯恩地图公司：投资组合带来可观的收入，公司没有任何财务压力，保险公司对地图价格满意，股东也能收到分红。
- 五、桑伯恩地图公司：保险公司总部的承保部门，凭借这些信息就能评估全国各地代理商承揽的业务。
- 五、桑伯恩地图公司：保险公司认为一图胜千言——通过地图，可以判断一项风险的定价是否合理、某个地区发生大规模火灾的概率、应采取何种再保险安排等。
- 五、桑伯恩地图公司：进入50年代初，一种名为"记分卡"（carding）的新承保方法问世，蚕食了桑伯恩的业务。
- 五、桑伯恩地图公司：我们的主要利润来源，仍然是买入[[低估]]的证券、待[[低估]]修复后卖出，以及投资于利润取决于公司行动而非市场走势的特殊情况。
- 五、桑伯恩地图公司：尽管"记分卡"已经兴起，仍有逾5亿美元的火险保费是通过"地图"承保方式承揽的。
- 五、桑伯恩地图公司：在我进入董事会之前，14位董事中有9位是保险界的知名人士，可在公司的105,000股流通股中，这9位合计只持有46股。

## 详细摘要

### 五、桑伯恩地图公司

投资组合带来可观的收入，公司没有任何财务压力，保险公司对地图价格满意，股东也能收到分红。保险公司总部的承保部门，凭借这些信息就能评估全国各地代理商承揽的业务。保险公司认为一图胜千言——通过地图，可以判断一项风险的定价是否合理、某个地区发生大规模火灾的概率、应采取何种再保险安排等。

### 年度概览

那一年，几乎所有投资公司都赚了钱，但其中能赶上或超过道指涨幅的不足10%。不过，我也说过：即便我们真的实现了长期超越指数的目标，也不要以为我们每年都能领先。如果这是真的，那说明我们的投资组合比持有蓝筹股的人更保守，尽管我们的投资方式肯定和大多数人截然不同。

### 四、预先存入资金

虽然我破过一次例，但凡是有多个合伙人的账户，年中修改合伙协议实在麻烦，所以对于这类账户，追加投资只能在年末进行。到年末，视合伙人修改协议的情况，预先存入的资金连同利息将转为合伙人资本，此后参与盈亏分享。

### 三、管理多个合伙账户

我尽量让所有账户"按照相似比例投资相同的股票"，但每年各账户之间还是会有一些差异。

> 其中几位所经营的保险公司持有少量桑伯恩股票，但在各自的投资组合里不过沧海一粟。

> 这意味着：1938年，在经济和股市都颇为萧条的时候，买入桑伯恩股票的人给地图业务的估值是每股90美元（110美元减去与地图业务无关的20美元投资价值）。
